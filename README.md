# C# Player's Guide Exercises

This repository contains my solutions to exercises from **C# Player's Guide**.

I am using this repository as a structured training ground to build strong fundamentals in C# and software development.

---

## Purpose

This is not just a collection of completed exercises.

It is part of a deliberate effort to:
- think in terms of **systems and logic**, not just syntax
- develop **discipline through consistent execution**
- build **clean, readable, and intentional code**
- strengthen my ability to **translate problems into working solutions**

---

## What I’m Training

Through these exercises, I am focusing on:

- Control flow (if/else, loops, branching)
- Method design and responsibility
- Understanding how types enforce rules on data
- Breaking problems into smaller steps
- Writing code that is clear and predictable
